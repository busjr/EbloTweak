## Использование

### Скачайте [архив](https://github.com/busjr/EbloTweak/releases) и распакуйте в удобном месте, укажите папку с расширением в:

> [!WARNING]
> Не удаляйте и не перемещайте эту папку после установки, иначе расширение перестанет работать

### Chrome
- Введите в адресную строку ```chrome://extensions/```
- Включите "Режим разработчика" (Developer mode) в правом верхнем углу
- Нажмите кнопку "Загрузить распакованное расширение" и выберите папку

### Firefox

> [!NOTE]
> В Firefox обычная установка распакованных папок через about:addons часто требует подписи дополнения. Для ручной установки лучше использовать отладочный режим:

#### Вариант А: Временная установка
- Введите в адресную строку ```about:debugging#/runtime/this-firefox```
- Нажмите кнопку "Загрузить временное дополнение..." (Load Temporary Add-on)
- Выберите файл manifest.json в папке с расширением

#### Вариант Б: Постоянная установка (zip)
- Перейдите в ```about:addons```
- Нажмите на шестеренку и выберите "Установить дополнение из файла..."
- Выберите заранее упакованный архив (zip) расширения
